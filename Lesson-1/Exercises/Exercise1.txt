Exercise 1.1
Compare the description of the program structure that we have just given with the program we gave when
describing the purpose of this lesson. What part of the program corresponds to DescriptiveComment? What
parts correspond to StandardHeaderFile and StatementSequence?


(1) DescriptiveComment corresponds to: //Displays Hello world on the screen

(2) StandardHeaderFile corresponds to: <iostream>

(3) StatementSequence corresponds to the statements inside the main function:

cout << "Hello world" << endl;
return 0;
